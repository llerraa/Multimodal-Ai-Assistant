# Multimodal-Ai
 AI-ассистент для автоматического создания тестовых вопросов на основе изображений.

**Архитектура и библиотеки:**
- Модуль анализа изображений (BLIP-large)
- Трансформер для перевода (NLLB-200)
- Библиотека PyTorch (используется для загрузки и запуска обеих моделей на графическом процессоре (GPU))
- Gradio (платформа для быстрого развертывания интерактивного веб-интерфейса, позволяющая пользователям загружать изображения и мгновенно видеть результаты)
